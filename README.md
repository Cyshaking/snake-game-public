# 贪吃蛇大冒险

一个现代化、功能丰富的贪吃蛇游戏实现，支持键盘和鼠标控制，拥有精美的视觉效果和流畅的游戏体验。

## 游戏特点

### 🎮 多样化控制方式
- **键盘控制**：支持方向键和WASD键移动，按住按键可加速
- **鼠标控制**：移动鼠标引导蛇头，提供更快的响应速度
- **操作灵活**：可随时切换控制方式，适应不同玩家习惯

### 🎨 现代化视觉设计
- **精美界面**：采用渐变背景、半透明效果和圆角设计
- **视觉反馈**：食物闪烁动画、蛇身渐变效果、网格背景
- **响应式布局**：游戏容器自适应，提供良好的视觉体验

### 🏆 游戏功能
- **计分系统**：实时显示当前得分和最高分
- **本地存储**：使用localStorage保存最高分记录
- **游戏控制**：开始、暂停/继续、重新开始等功能按钮
- **暂停功能**：空格键快速暂停/继续游戏
- **游戏结束画面**：清晰的游戏结束提示和最终得分展示

### ⚡ 流畅的游戏体验
- **多级速度**：普通速度、加速速度和鼠标模式速度
- **智能控制**：蛇不会180度转向，提供更合理的游戏体验
- **视觉提示**：暂停和游戏结束时的遮罩提示

## 安装说明

1. 克隆或下载本仓库到本地
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

无需任何额外依赖，纯前端实现，支持现代浏览器。

## 操作指南

### 键盘控制
- **方向键**：控制蛇的移动方向（上、下、左、右）
- **WASD键**：同样可以控制蛇的移动方向
- **按住方向键/WASD键**：蛇的移动速度提高
- **空格键**：暂停/继续游戏

### 鼠标控制
- 将鼠标移动到游戏区域内，蛇会自动追踪鼠标方向
- 鼠标模式下，蛇的移动速度比键盘模式更快
- 移出鼠标后，自动切换回键盘控制模式

### 游戏按钮
- **开始游戏**：开始或继续游戏
- **暂停/继续**：暂停当前游戏或继续暂停的游戏
- **重新开始**：重置游戏状态，重新开始

## 游戏规则

1. 控制蛇头移动，吃掉随机生成的食物
2. 每吃掉一个食物，蛇的长度增加，得分增加
3. 蛇头碰到边界或自身身体时，游戏结束
4. 蛇不能直接180度转向（例如向右移动时不能直接向左转）
5. 尝试获得最高分数！

## 技术实现

- **HTML5 Canvas**：用于游戏渲染
- **JavaScript**：实现游戏逻辑和交互
- **CSS3**：创建现代化的游戏界面和动画效果
- **localStorage**：保存最高分记录

## 游戏优化

- **性能优化**：减少不必要的重绘，确保流畅运行
- **响应式设计**：适应不同屏幕尺寸
- **用户体验**：提供清晰的视觉反馈和操作提示

## 开发背景

这是一个基于经典贪吃蛇游戏的现代重制版，保留了原始游戏的趣味性，同时添加了现代化的视觉效果和多种控制方式，旨在为玩家提供更好的游戏体验。
